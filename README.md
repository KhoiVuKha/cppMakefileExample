# cppMakefileExample

- E.g.1:
    1. Build code:
      make
    2. Run:
      ./output
    3. Clean build artifacts:
      make clean

- E.g.2:
    1. Build code:
      make
    2. Run:
      ./output
    3. Clean build artifacts:
      make clean

- E.g.3:
    1. Build code:
      make build
    2. Format code:
      make format
    3. Clean build artifacts:
      make clean
